User
What is machine learning?

Assistant
Machine learning is a subset of artificial intelligence that enables systems to learn and improve from experience without being explicitly programmed.

User
Can you give me an example?

Assistant
Sure! A common example is email spam filtering. The system learns to identify spam emails by analyzing patterns in messages marked as spam by users, then applies those patterns to new incoming emails.

User
How does it differ from traditional programming?

Assistant
In traditional programming, you write explicit rules and logic. In machine learning, you provide data and let the algorithm discover the patterns and rules automatically. This is more flexible for complex problems where rules are hard to define manually.
